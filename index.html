project 04 // ha kubernetes on aws

TaskApp — kOps Capstone

kOps · Terraform · Multi-AZ · cert-manager · AWS Secrets CSI Driver
3-AZ HA topology Spot workers Auto etcd backups every 15min

// what shipped

  • Migrated a locally-run TaskApp (React + Flask + Postgres) to a highly available, kOps-managed Kubernetes cluster spanning 3 Availability Zones.
  • Terraform-provisioned everything from scratch: VPC, public/private subnets, S3 remote state with DynamoDB locking, and automated billing alerts.
  • Private Kubernetes topology with a multi-master control plane and Spot-instance worker nodes for cost efficiency without sacrificing HA.
  • Automated HTTPS end-to-end via cert-manager and NGINX ingress, with Route53 handling DNS delegation for frontend and backend hosts.
  • Replaced the local Postgres container with a private-subnet RDS instance, with secrets pulled through the AWS Secrets Store CSI driver — nothing sensitive lives in a manifest.
